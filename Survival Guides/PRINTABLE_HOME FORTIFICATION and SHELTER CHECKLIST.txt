HOME FORTIFICATION & SHELTER CHECKLIST
Goal: Harden the structure against weather and intruders.
--------------------------------------------------------------------------------

PERIMETER & ENTRY
[ ] Reinforce door strikes with 3-inch screws.
[ ] Block ground-level windows with heavy furniture or cut plywood.
[ ] Clear yard of objects that can be thrown/used as tools by intruders.
[ ] Set up "Noise Traps" (bells/cans/gravel) at entry points.

INTERIOR SAFETY
[ ] Designate "Safe Room" (thick walls, no windows).
[ ] Clear a path to the Safe Room for rapid retreat.
[ ] Place fire extinguishers (ABC type) on every floor.
[ ] Verify Carbon Monoxide detectors have battery backup.

UTILITIES & CLIMATE
[ ] Locate main water shut-off valve.
[ ] Locate main gas shut-off valve (and wrench).
[ ] Locate main electrical breaker.
[ ] Insulate windows (Bubble wrap/blankets) if heating fails.
[ ] Set up sleeping area off the floor (mattresses on pallets/tables).

SANITATION ZONES
[ ] Designate a waste disposal area (Latrine) at least 200ft from water.
[ ] Stockpile heavy-duty trash bags for waste.
[ ] Set up a handwashing station (bucket + soap/ash) at entry.
